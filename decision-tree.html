Base learner

A decision tree is greedy partitioning of feature space.

At every node the tree asks one question, chooses the split that most improves the target purity, and repeats until stopping rules say the leaves are specific enough.

Interview answer Split, stop, average.

Classification leaves vote by class distribution; regression leaves predict the mean target.

Core mechanics

What happens at a node?

Candidate split

For a numerical feature, sort values and test thresholds between adjacent values. For categorical data, test grouped categories or use an encoding strategy.

Objective

Choose the split that gives the biggest weighted impurity reduction. The choice is local, so the tree is greedy, not globally optimal.

Leaf prediction

Classification uses class frequencies or the majority class. Regression uses the mean target in the leaf.

Regularization

Limit max depth, min samples per leaf, min impurity decrease, max leaves, or prune after training.

Formulas

Know these well enough to derive them aloud.

Gini impurity Gini(S) = 1 - sum_k p_k^2

Probability that a random label would be misclassified if sampled from the node distribution.

Entropy H(S) = -sum_k p_k log(p_k)

Uncertainty in the class distribution. Information gain is parent entropy minus child entropy.

Regression split gain = SSE(parent) - SSE(left) - SSE(right)

Equivalent to reducing variance inside leaves.

Drills

Questions you should answer in one minute.

Why does a deep tree overfit?

Give the short answer first, then reveal the interview version.

Answer

A deep tree can isolate tiny groups or individual samples, so training impurity keeps falling while the learned boundaries become high-variance and unstable.

Gini or entropy?

Explain whether the choice usually changes model quality.

Answer

They often choose similar splits. Entropy has information-theoretic meaning; Gini is simpler and slightly cheaper. Regularization usually matters more.

Traps

Common weak answers.

“Trees do not need preprocessing.”

They are insensitive to monotonic scaling, but categorical encoding, missing values, and leakage still matter.

“Feature importance is always reliable.”

Impurity importance can favor high-cardinality and frequently split features.

“A tree extrapolates.”

A regression tree predicts leaf averages, so outside the observed range it stays piecewise constant.
