What to look for

Initialization changes the computation graph that backprop sees.

Backprop is local multiplication. If a unit is copied from another unit, both receive the same signal. If its derivative is zero, the signal stops. If all weights are tiny, useful differences are weak. Good initialization keeps units different and gradients measurable.

Zeros and copied units

Hidden units remain interchangeable, so capacity is wasted.

Dead ReLU

Negative pre-activations give derivative zero, blocking first-layer updates.

Saturated tanh

Outputs near -1 or 1 have tiny derivatives, so learning crawls.

Healthy random

Units start different, active, and gradient-connected.
