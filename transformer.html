Current idea

State is the main object.

Every token position carries a vector. That table of vectors is the residual stream.

Intuition

The stream is a shared notebook.

Each layer reads the notebook and writes a small correction into it.

What changes as depth increases?

Component contracts

What each piece is responsible for.

Residual stream

Job: carry the current token-state table.

Why: gives every block a shared workspace.

Without it: deep layers lose earlier information and train poorly.

Layer norm

Job: make vectors readable at a stable scale.

Why: keeps repeated blocks numerically calm.

Without it: activations drift and deep training becomes fragile.

Self-attention

Job: move information between positions.

Why: distant tokens need to influence each other.

Without it: positions mostly think alone.

Attention heads

Job: run several communication patterns at once.

Why: syntax, references, and task cues may need separate routes.

Without them: fewer independent channels per layer.

MLP

Job: transform each position locally.

Why: attention gathers context; MLP turns it into features.

Without it: the model communicates but computes weakly.

Position information

Job: tell the model where tokens are.

Why: order changes meaning.

Without it: many reorderings look too similar.

Causal mask

Job: hide future tokens in decoder models.

Why: generation only knows the past.

Without it: training can cheat.

Output head

Job: turn the final state into vocabulary scores.

Why: it reads internal state into a next-token distribution.

Without it: no token probabilities come out.

Misconceptions

Common wrong pictures.

Attention is the whole model.

Attention is communication. The residual stream and MLPs are equally central.

Attention weights are explanations.

They show routing, not what information was written or how later layers used it.

Each token stores one clean fact.

A vector is a superposition of many features at once.

Layers all do the same thing.

The operation repeats, but the state being operated on becomes richer.

The MLP is optional decoration.

It is the main local nonlinear computation inside each block.

Meaning lives in one place.

It is distributed across positions, layers, heads, and vector directions.
