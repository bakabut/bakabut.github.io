Gradient boosting

XGBoost adds trees that fix the current model’s mistakes.

The model is an additive ensemble. Each new tree is chosen to reduce the training objective, using gradient and second-order curvature information plus explicit regularization.

Interview answer Optimize loss with trees.

Random forest averages independent trees; XGBoost builds trees sequentially.

Core mechanics

The boosting loop.

Current prediction

Start with a constant prediction, then repeatedly add a tree to improve the objective.

Gradients

For each row, compute how the loss changes if the prediction changes. This tells the next tree what direction to correct.

Second order term

XGBoost also uses Hessians, giving a local quadratic approximation to the loss and better leaf weight estimates.

Regularized tree

The objective penalizes too many leaves and large leaf weights, so a split must earn its complexity.

Formulas

Interview-level math.

Additive model ŷ_i = sum_t f_t(x_i)

Each tree contributes a score; the final prediction is the sum transformed by the task link function.

Regularized objective Obj = loss(y, ŷ) + sum_t Ω(f_t)

The regularizer penalizes tree complexity and leaf weights.

Leaf weight intuition w* = -G / (H + λ)

A leaf moves opposite the summed gradient, damped by curvature and L2 regularization.

Drills

Answer these without listing hyperparameters first.

Why does boosting reduce bias?

Answer

Each new tree is trained to improve the remaining errors of the current ensemble, so the model can build a strong predictor from many weak corrections.

How do you control overfitting?

Answer

Use learning rate, early stopping, max depth, min child weight, row and column subsampling, L1/L2 regularization, and a validation split that matches deployment.

Traps

Common mistakes.

“Boosting trains trees independently.”

No. Later trees depend on current predictions, so training is sequential.

“More estimators is always better.”

More rounds can overfit unless learning rate, regularization, and early stopping are handled.

“Residuals only work for MSE.”

For general losses, talk about gradients. Squared error residuals are one special case.
